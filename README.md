# pgn-editor

Chess editor to easily create PGN files.

<!-- omit in toc -->
## Table of Contents

- [Usage](#usage)
- [License](#license)

## Usage

Install dependencies
```bash
npm install
```

Build bundle
```bash
npm run build -- --mode production
```

Open `./dist/index.html` in a browser.

## Development

Install dependencies
```bash
npm install
```

Build & start server
```bash
npm start
```

Open http://localhost:8080/
